Configuration Firebase

Pour connecter cette application à Firebase, suivez ces étapes :

  1. Créez un projet sur Firebase Console
  2. Activez Authentication et Firestore Database
  3. Copiez votre configuration Firebase et remplacez le code ci-dessous
  4. Déployez l'application en ligne (voir instructions plus bas)

// Remplacez cette configuration par la vôtre
const firebaseConfig = {
    apiKey: "VOTRE_CLE_API",
    authDomain: "votre-projet.firebaseapp.com",
    projectId: "votre-projet",
    storageBucket: "votre-projet.appspot.com",
    messagingSenderId: "123456789",
    appId: "VOTRE_APP_ID"
};
